# BilgeProject

## Description

BilgeProject is a modern web platform designed to preserve and share cultural heritage through technology. The platform combines sacred places, calendar events, and live stories to deliver cultural heritage to future generations.

The application features a responsive design with sections for hero content, inspirational quotes, places exploration, and an interactive chatbot, all built with React and styled using Tailwind CSS.

## Features

- **Hero Section**: Engaging landing area with gradient backgrounds and animated elements
- **Quote Section**: Inspirational quotes about cultural preservation
- **Places Section**: Showcase of sacred and historical places
- **Interactive Chatbot**: AI-powered assistant for user interactions
- **Responsive Design**: Optimized for desktop and mobile devices
- **Modern UI**: Clean, accessible interface with smooth animations

## Tech Stack

- **Frontend Framework**: React 18.3.1
- **Build Tool**: Vite 5.4.10
- **Styling**: Tailwind CSS 3.4.14
- **CSS Processing**: PostCSS 8.4.47 with Autoprefixer 10.4.20
- **Language**: JavaScript (ES Modules)

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd front
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally
